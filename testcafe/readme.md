## Setup

#### requirements

* node ^13.1.0
* testcafe
* yarn

#### installation

```
$brew install node
$brew install yarn 
$npm install -g testcafe
$yarn 
```

## Usage

1. Register at dashboard.evrythng.com
2. Update config.js with your credentials
3. run:

```
$cd Actions/ && testcafe chrome thng.js
```
